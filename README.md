1️⃣ 하루 세 문제 챌린지

	매일 AI🤖 가 제공하는 세 문제로 코드 실력을 쌓아가세요❗️
 	초급자도 쉽게하는 코딩테스트 준비, 우리와 함께 시작하면 성공까지 단축 ⏰

2️⃣ AI 추천 코드로 1차 피드백

	AI가 제시하는 모범 답안으로 문제의 핵심을 파악하세요 🎯

3️⃣ 내 코드에 대한 AI 코드 리뷰

	내 코드, AI가 꼼꼼히 리뷰해 드립니다 🖍️
	실수는 줄이고, 실력은 높이는 개인 맞춤형 피드백을 경험하세요❗️
	(📣 해당 서비스는 하루 1회로 제한됩니다)

4️⃣ 레벨별 캐릭터 성장 시스템

	귀여운 캐릭터와 함께 성장하는 코딩 여정❗️
	문제를 풀며 경험치를 쌓아, 캐릭터를 성장시키세요 🎮

5️⃣ 월간 달성률 확인 기능

	이 달의 코딩 성과를 한눈에 🗓️
	월간 달성률로 자신의 성장을 확인하고, 더 큰 목표를 향해 나아가세요 🏃

6️⃣ 문제 별 코드 저장 및 북마크 기능

	중요한 코드, 북마크 하세요 📌
	나중에 다시 볼 수 있도록 내 코드를 저장하고, 언제든지 쉽게 복습할 수 있어요 👀
